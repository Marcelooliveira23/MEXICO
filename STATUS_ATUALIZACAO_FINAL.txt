┌────────────────────────────────────────────────────────────────────────────────┐
│ 📞 TROUBLESHOOTING RÁPIDO                                                     │
└────────────────────────────────────────────────────────────────────────────────┘

"Can't connect to MySQL"
→ Verificar: mysql -u root -p -e "SELECT 1"
→ Se erro: Instalar MySQL ou iniciar serviço

"Redis connection refused"
→ Verificar: redis-cli ping
→ Se erro: docker run -d -p 6379:6379 redis:7-alpine

"ModuleNotFoundError: No module named 'flask'"
→ Verificar: venv ativo (deve ter (venv) no prompt)
→ Executar: pip install -r requirements_modernized.txt

"Port 5000 already in use"
→ Matar processo: taskkill /PID <PID> /F
→ Ou usar porta diferente: PORT=5001 python app_modernized_v2.py

Mais detalhes: Ver GUIA_INICIO_RAPIDO.md seção "Troubleshooting Comum"
