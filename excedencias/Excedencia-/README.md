## 🚀 Início Rápido

```powershell
# 1. Criar ambiente virtual
python -m venv venv

# 2. Ativar ambiente
.\venv\Scripts\activate

# 3. Instalar dependências
pip install -r requirements.txt

# 4. Executar protótipo
python src/main.py
```
